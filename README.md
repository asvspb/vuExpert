# Vue-SQLite-Redis-SCSS Образовательный Проект

Этот проект представляет собой образовательное веб-приложение, разработанное с использованием современных технологий, включая Vue.js 3, FastAPI, SQLite, Redis и SCSS. Проект предназначен для изучения и практики современных подходов к разработке веб-приложений.

## Архитектура проекта

Проект разделен на следующие основные компоненты:

### Frontend (Клиентская часть)
- **Vue.js 3** - Современный фреймворк для создания пользовательских интерфейсов
- **Vue Router** - Для навигации между страницами
- **Pinia** - Для управления состоянием приложения
- **Element Plus** - Компонентная библиотека для Vue
- **SCSS** - Для стилизации компонентов
- **Vite** - Быстрая сборка и разработка
- **Vitest** - Для модульного тестирования
- **Playwright** - Для сквозного (e2e) тестирования

### Backend (Серверная часть)
- **FastAPI** - Современный веб-фреймворк для создания API с высокой производительностью
- **SQLite** - Встраиваемая реляционная база данных
- **Redis** - Система хранения данных в памяти (key-value)
- **Poetry** - Для управления зависимостями Python

## Запуск проекта

### Предварительные требования
- Node.js (для frontend)
- Python 3.8+ (для backend)
- Poetry (для управления зависимостями Python)
- Docker и Docker Compose (опционально)

### Локальный запуск

#### Frontend:
```bash
cd vuExpert
npm install
npm run dev
```

#### Backend:
```bash
cd vuExpert/backend
poetry install
poetry run python -m app.main
```

### Запуск с Docker:
```bash
docker-compose up --build
```

## Тестирование

### Frontend:
- Модульные тесты: `npm run test`
- Сквозные тесты: `npm run e2e`

### Backend:
- Запуск тестов: `poetry run pytest`

## Стиль кодирования

- Для JavaScript/TypeScript/Vue используется ESLint
- Для SCSS используется Stylelint
- Для Python используется форматирование через Black и линтер через Flake8
