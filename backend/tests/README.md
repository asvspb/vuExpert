# Директория backend/tests

Эта директория содержит тесты для серверного приложения на FastAPI.

## Структура

- `test_main.py` - Тесты для основного приложения и базовых маршрутов (health, users, logs)
- `test_crud.py` - Тесты для операций CRUD (users)
- `conftest.py` - Настройки фикстур pytest (база данных, сессии, http-клиент)

## Запуск

Используйте команду `pytest` (или `poetry run pytest`) для выполнения изолированных асинхронных тестов.
