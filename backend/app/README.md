# Директория backend/app

Эта директория содержит основной код серверного приложения на FastAPI.

## Структура

- `main.py` - Входная точка приложения, инициализация FastAPI, подключение роутеров и CORS
- `models.py` - Модели базы данных (SQLAlchemy)
- `schemas.py` - Схемы Pydantic для валидации данных (входящие/исходящие)
- `crud.py` - Функции для выполнения операций с БД (Create, Read, Update, Delete)
- `database.py` - Настройка асинхронного движка SQLAlchemy и сессий
- `settings.py` - Конфигурация приложения через pydantic-settings
- `routers/` - Эндпоинты API, разбитые по доменным областям
