# Директория backend

Эта директория содержит серверную часть приложения, реализованную на FastAPI. FastAPI - это современный веб-фреймворк для создания API с высокой производительностью. В качестве основной базы данных используется PostgreSQL, а для кеширования — Redis.

## Структура

- `app/` - Основной код приложения
  - `main.py` - Входная точка приложения
  - `models.py` - Модели базы данных (SQLAlchemy)
  - `schemas.py` - Pydantic схемы для валидации данных
  - `crud.py` - Операции CRUD (создание, чтение, обновление, удаление)
  - `database.py` - Конфигурация базы данных (PostgreSQL через asyncpg/psycopg2)
- `tests/` - Тесты для приложения (Pytest)
- `Dockerfile` - Файл для создания Docker образа
- `requirements.txt` - Зависимости Python
- `poetry.lock` и `pyproject.toml` - Конфигурации для управления зависимостями (Poetry)
