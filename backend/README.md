# Директория backend

Эта директория содержит серверную часть приложения, реализованную на FastAPI. В качестве базы данных используется PostgreSQL, а для кеширования и ограничения запросов — Redis.

## Структура

- `app/` - Основной код приложения
  - `main.py` - Входная точка приложения
  - `routers/` - Эндпоинты приложения, разбитые по доменам (health, counter, items, users, logs)
  - `models.py` - Модели базы данных (SQLAlchemy 2.0 DeclarativeBase)
  - `schemas.py` - Pydantic схемы для валидации данных
  - `crud.py` - Операции CRUD
  - `database.py` - Асинхронная конфигурация базы данных
  - `settings.py` - Управление конфигурацией (pydantic-settings)
  - `redis_client.py` и `limiter.py` - Настройки Redis и rate-limiting (slowapi)
- `tests/` - Тесты для приложения (Pytest, изолированные асинхронные тесты)
- `Dockerfile` - Файл для создания Docker образа
- `poetry.lock` и `pyproject.toml` - Конфигурации для управления зависимостями (Poetry)
- `.env.example` - Шаблон файла переменных окружения
