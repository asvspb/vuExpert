# Директория src/types

Эта директория содержит определения типов TypeScript для проекта.

## Структура

В этой директории могут находиться файлы, содержащие:
- Общие типы и интерфейсы, используемые в проекте
- Определения типов для API запросов и ответов
- Типы для состояния приложения
- Глобальные типы и интерфейсы

## Текущие файлы

- `index.ts` - Основные типы и интерфейсы проекта
- `api.ts` - Типы для API запросов и ответов
