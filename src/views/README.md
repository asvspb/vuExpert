# Директория src/views

Эта директория содержит компоненты-страницы (Views) маршрутизатора Vue. 

Каждый `.vue` файл здесь представляет отдельный URL-маршрут в приложении.

## Структура

- `HomeView.vue` - Дашборд стартового шаблона
- `AboutView.vue` - Описание стека
- `NotFoundView.vue` - Страница ошибки 404
