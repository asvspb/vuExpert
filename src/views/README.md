# Директория src/views

Эта директория содержит представления (views) приложения, которые представляют собой страницы или маршруты.

## Структура

В этой директории могут находиться файлы, содержащие:
- Основные страницы приложения
- Компоненты, которые соответствуют конкретным маршрутам
- Страницы, которые используют общие макеты и компоненты

## Текущие файлы

- `VuetifyDemo.vue` - Демонстрационная страница с примерами компонентов Vuetify
