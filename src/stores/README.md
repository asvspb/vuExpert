# Директория src/stores

Эта директория содержит хранилища состояния (stores) для управления состоянием приложения с использованием Pinia.

## Структура

- `index.ts` - Главный файл для экспорта всех stores
- `counter.ts` - Хранилище для управления состоянием счетчика
- `app.ts` - Глобальное хранилище состояния приложения

## Текущие файлы
