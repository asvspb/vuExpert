# Директория src/stores

Эта директория содержит хранилища состояния приложения, реализованные с использованием Pinia - официального решения для управления состоянием в Vue.js приложениях.

## Структура

- `app.js` - Основное хранилище приложения (app store)

## Описание

Файл `app.js` определяет основное хранилище приложения с использованием Pinia. Хранилище содержит состояние, геттеры и действия, которые могут быть использованы в компонентах Vue для управления и доступа к общему состоянию приложения.

### Состояние (state)

- `message` - строка с приветственным сообщением
- `counter` - числовое значение счетчика
- `darkMode` - булевое значение для переключения темной темы

### Геттеры (getters)

- `doubleCounter` - возвращает удвоенное значение счетчика

### Действия (actions)

- `increment()` - увеличивает значение счетчика на 1
- `reset()` - сбрасывает значение счетчика в 0
- `toggleDarkMode()` - переключает состояние темной темы
- `setMessage(msg)` - устанавливает новое значение сообщения
