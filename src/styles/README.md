# Директория src/styles

Эта директория содержит файлы стилей для клиентской части приложения.

## Структура

- `main.scss` - Основной файл стилей
- `variables.scss` - Переменные стилей (цвета, размеры и т.д.)
- `mixins.scss` - SCSS миксины для повторного использования стилей

## Текущие файлы

- `main.scss` - Основной файл стилей
- `variables.scss` - Переменные стилей (цвета, размеры и т.д.)
- `mixins.scss` - SCSS миксины для повторного использования стилей

## Используемые шрифты

В проекте используются следующие шрифты из Google Fonts:

- **Roboto** - основной шрифт для текста
- **Montserrat** - шрифт для заголовков
- **Roboto Mono** - моноширинный шрифт для кода

Шрифты подключаются через CSS-импорт в файле `main.scss` и определяются в переменных в файле `variables.scss`.
