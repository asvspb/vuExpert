# Директория styles

Директория `styles` содержит файлы стилей, написанные на SCSS, которые используются в приложении.

## Описание файлов

- `main.scss` - основной файл стилей, который может содержать импорты других стилевых файлов и глобальные стили
- `mixins.scss` - файл с миксинами SCSS, которые можно использовать для повторного использования стилевых фрагментов
- `variables.scss` - файл с переменными SCSS, содержащий цвета, размеры и другие параметры для стилизации

## Назначение

Эта директория отвечает за стилизацию приложения. Использование SCSS позволяет создавать более гибкие и поддерживаемые стили с возможностью использования переменных, миксинов и других функций препроцессора.
