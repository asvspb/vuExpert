# Директория src

Эта директория содержит клиентскую часть приложения на Vue.js.

## Структура

- `App.vue` - Главный компонент приложения
- `main.js` - Входная точка клиентского приложения
- `components/` - Компоненты Vue.js
- `composables/` - Composables (логика, доступная для использования в компонентах)
- `layouts/` - Макеты и компоновка страниц
- `router/` - Конфигурация маршрутизации
- `services/` - Сервисы для взаимодействия с API
- `stores/` - Хранилища состояния (Pinia)
- `styles/` - Файлы стилей
- `views/` - Представления (страницы) приложения
