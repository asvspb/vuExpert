# Директория src

Директория `src` содержит весь исходный код приложения Vue.js.

## Описание файлов

- `App.vue` - главный компонент приложения, который служит точкой входа для пользовательского интерфейса
- `main.js` - точка входа в приложение Vue, где создается экземпляр приложения и монтируется к DOM
- `components/` - директория с компонентами Vue
- `styles/` - директория со стилями SCSS

## Назначение

Эта директория содержит все исходные файлы, необходимые для работы приложения. Все компоненты, стили и логика приложения находятся здесь.
