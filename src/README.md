# Директория src

Директория `src` содержит весь исходный код приложения Vue.js.

## Описание файлов

- `App.vue` — главный компонент приложения, корневой блок должен соответствовать BEM (например, `.app`)
- `main.js` — точка входа в приложение Vue
- `components/` — директория с Vue-компонентами (SFC). Используем `<script setup>`, Composition API, BEM и `<style scoped>`
- `styles/` — SCSS-стили, переменные и миксины (`src/styles/variables.scss`, `src/styles/mixins.scss`)

## Назначение

Эта директория содержит все исходные файлы, необходимые для работы приложения. Все компоненты, стили и логика приложения находятся здесь.
