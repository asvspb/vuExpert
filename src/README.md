# Директория src

Эта директория содержит клиентскую часть приложения на Vue 3.

## Структура

- `App.vue` - Главный компонент приложения
- `main.js` - Входная точка клиентского приложения
- `components/` - Переиспользуемые UI-компоненты
- `views/` - Страницы маршрутизации (Home, About, NotFound)
- `composables/` - Выделенная логика
- `useApi.js` — базовая обертка для HTTP-клиента (интерцепторы).
- `useTitle.js` — пример композибла для смены заголовка страницы.
- `services/` - Сервисы для взаимодействия с API (`http.js`, `api.js`)
- `router/` - Конфигурация Vue Router
- `stores/` - Управление состоянием (Pinia)

## Архитектура и стилизация

Проект использует **Tailwind CSS v4** для стилизации компонентов. Мы полностью отказались от SCSS и BEM в пользу Utility-First подхода. Файл `tailwind.css` подключается глобально в `main.js`.

Также проект использует `dependency-cruiser` для контроля архитектурных слоев (защита UI от доменной логики и предотвращение циклических зависимостей). 
Команды для работы:
- `npm run lint:deps` — проверка нарушений архитектурных правил.
- `npm run depcruise:graph` — генерация SVG-графа зависимостей фронтенда (требует установленного `graphviz`).
