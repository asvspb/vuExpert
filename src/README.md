# Директория src

Эта директория содержит клиентскую часть приложения на Vue.js.

## Структура

- `App.vue` - Главный компонент приложения
- `main.js` - Входная точка клиентского приложения
- `components/` - Компоненты Vue.js
- `composables/` - Composables (логика, доступная для использования в компонентах)
- `services/` - Сервисы для взаимодействия с API
- `styles/` - Файлы стилей

## Архитектура и зависимости

Проект использует `dependency-cruiser` для контроля архитектурных слоев (защита UI от доменной логики и предотвращение циклических зависимостей). 
Команды для работы:
- `npm run lint:deps` — проверка нарушений архитектурных правил.
- `npm run depcruise:graph` — генерация SVG-графа зависимостей фронтенда (требует установленного `graphviz`).
