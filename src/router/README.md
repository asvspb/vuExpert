# Директория src/router

Эта директория содержит файлы маршрутизации для приложения Vue.js. В текущем проекте используется Vue Router для навигации между различными страницами и компонентами приложения.

## Структура

- `index.js` - Основной файл конфигурации маршрутов приложения

## Описание

Файл `index.js` определяет маршруты приложения с использованием lazy-loading (ленивой загрузки) для оптимизации производительности. Это позволяет загружать только те компоненты, которые необходимы для отображения текущей страницы.

### Маршруты

- `/` - Главная страница приложения (HomeView)
- `/docs` - Страница документации (Documentation)
- `/db` - Пример работы с базой данных (DatabaseExample)
- `/hello` - Тестовый компонент (HelloWorld)

Каждый маршрут имеет метаданные с заголовком страницы, которые используются для динамического изменения заголовка в браузере.

Также реализована функция `scrollBehavior` для автоматического прокручивания к верху страницы при переходе между маршрутами.
