# Директория src/layouts

Эта директория содержит макеты (layouts) для приложения, которые определяют общую структуру и компоновку страниц.

## Структура

В этой директории могут находиться файлы, содержащие:
- Основные макеты приложения
- Компоненты компоновки, которые используются на нескольких страницах
- Общие шаблоны для различных разделов приложения
