# Директория src/composables

Эта директория содержит Vue composables - функции, которые инкапсулируют и повторно используют логику с состоянием.

## Структура

В этой директории находятся файлы, содержащие:
- Логику состояния, доступную для использования в компонентах
- Функции для взаимодействия с API
- Утилиты для обработки данных
- Состояния, доступные на уровне приложения
