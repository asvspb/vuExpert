# Директория components

Директория `components` содержит компоненты Vue, которые используются в приложении.

## Описание компонентов

- `DatabaseConnectionExample.vue` - компонент с примером подключения к базе данных
- `DatabaseExample.vue` - компонент с примером работы с базой данных
- `Documentation.vue` - компонент для отображения документации

## Назначение

Компоненты в этой директории представляют собой переиспользуемые элементы пользовательского интерфейса, каждый из которых отвечает за определенную функциональность приложения. Они могут быть импортированы и использованы в других частях приложения.
